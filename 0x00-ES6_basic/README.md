# ES6 Basics Practice

This repository contains practice exercises for ES6 basics.

## Overview

The exercises in this repository cover various ES6 features including:

- `let` and `const`
- Arrow functions
- Template literals
- Default parameters
- Destructuring assignment
- and more...
